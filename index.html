Nova versão UI/UX à caminho! Junho/2022 Ver-1.5.0

AndroidUserTools é um programa desenvolvido em C/CPP que dentre outras coisas, auxilia na modificação do seu aparelho android, por meio dele é possível instalar o TWRP, Desbloquear o Bootloader, Instalar e desinstalar apps via ADB/Fastboot automaticamente!

Este programa está em desenvolvimento, é recomendado o download e a utilização da versão mais recente disponível, que é a 1.4.9, caso tenha problemas ao executar, consulte o nosso suporte através do github, é importante que o usuário possa nos passar algum feedback para que o programa possa ser otimizado constantemente.

O que é o projeto UI?

Atualmente o programa roda através do console do windows, o projeto UI ou User Interface, vai trazer ao usuário uma interface gráfica mais amigável ao usuário e com mais recursos, utilizando a programação orientada a objetos com o Qt, é planejado trazer uma versão do programa mais robusta e intuitiva.

As versões convencionais serão perdidas?

Não, todas as versões lançadas até o momento ainda serão disponibilizadas, embora a recomendação seja sempre que o usuário utilize as versões mais recentes do software!

As versões convencionais ainda serão atualizadas?

Somente para correção de futuros bugs, nenhuma outra função de alta escala será adicionada as versões executadas por meio do prompt de comando do Windows.

Existe alguma previsão para o lançamento?

O Projeto já foi estudado e agora está em fase de desenvolvimento e posteriormente em fase de testes, isso demanda um certo período de tempo, como este projeto é tocado somente por um desenvolvedor ainda, não há garantia de datas, no entanto é certo que estará finalizado ao menos no primeiro semestre de 2022, até Junho.

Tutorial

Você pode baixar a versão mais recente do programa através da aba versão ou caso queira, pode baixar através do github (atenção ao baixar versões antigas) algumas versões antigas podem dar erro de dll's quando são executadas, neste caso consulte a parte [1] que trata deste erro e trás uma resolução.

Assim que você tiver completado o download, abra o arquivo que estará em formato de zip com o winrar ou o descompactador que preferir, recomendamos o winrar pela sua praticidade feito isso basta extrair o arquivo para a sua área de trabalho ou qualquer que seja o arquivo.

Abra a pasta extraída e execute o programa AndroidUserTools.exe com previlégios, feito isso procure e selecione a opção que você precise utilizar. NOTA: para todos os métodos existe uma breve explicação de como se deve proceder para que você obtenha sucesso durante o processo.

[1] Caso esteja tendo problemas para baixar ou executar o programa, aqui vai algumas soluções, se estiver com o problema de dll geralmente o programa vai apitar um erro da falta de algumas dll's, esse problema já foi resolvido porém caso volte a aparecer baixe as dll's disponibilizadas no git, na pasta ´DLL'S´ e cole os no diretório raiz do programa AndroidUserTools/aqui
